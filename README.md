# MCOBrowserDiversity
This repository contains the code and data used as part of the manuscript "Global faunal response among herbivorous mammals to mid-Miocene changes in atmospheric carbon dioxide". 

**Bootstrapping**

The data required for the bootstrapping analysis and plots are included in the **Faunal lists** folder. In this folder is the code used for the analysis (*Bootstrapping code.R*) and the outputs containing the confidence intervals for each region. This analysis and code is adapted from a version originally written by John Damuth in 2020. 

**Faunal lists**

This folder contains the faunal lists for South America (*South American faunal lists.xslx*) and the Riversleigh World Heritage Site, Australia (*Riversleigh faunal lists.xlsx*). 

**Number of browsers**

This folder contains the code used to produce the plots for each region, except for the bootstrapping analysis (*Plots code.R*). In addition, it includes all the input data used for analyses. *Additional North American localities.xlsx* includes a list of the new localities from the North American Great Plains included beyond those of Janis et al. (2000, 2002, 2004). *NOW Faunal Data.xlsx* includes the data for all localities extracted from the NOW Database of Fossil Mammals. This spreadsheet also includes information about the changes in age assignments between 2014 and 2026. For Europe, Balkanotolia, Central Asia, the Indian Subcontinent, Eastern Africa, the North American Great Plains, and South America, two files are available: 

*Region_brachydonts.csv* - this includes age assignments, location and number of brachydonts and hypsodonts for each locality in each region

*Region_max bins.csv* - this includes the maximum number of brachydonts from that region for each time bin during the period of study

Two spreadsheets are also included for Riversleigh: 

*Riversleigh herbivores.csv* includes the number of herbivores of each type (arboreal browser, terrestrial browser and hypsodont - assumed grazer) for each fauna

*Riversleigh year comparisons.csv* includes the same data but comparing the lists between 2011 and 2026 to assess the "10 years in the library" effect for Riversleigh faunal lists (Sepkoski 1993).
